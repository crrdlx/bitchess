About

Bitchess is a fully autonomous chess game played between two blockchains. Bitcoin plays orange pieces. Ethereum plays silver pieces.

Every move is determined by the next block hash from that chain — no humans, no AI, just cryptographic entropy. The same starting block numbers always produce the same game, forever.

Game outcomes will eventually post to Nostr.

Rules & replicability

Who goes first: The last significant hex digit of a deciding Bitcoin block hash — odd = Bitcoin, even = Ethereum.

Move selection: Strip leading zeros (proof-of-work, not random). Characters 1–2 of the remaining hash select which piece to move. Characters 3–4 select the destination square. Illegal moves fall back further along the hash. A block that yields nothing is skipped.

Replicability: Block hashes are immutable on-chain. Starting a game at BTC block 840,000 today and again in ten years produces the exact same game. Every move log entry links to the block on its chain explorer for independent verification.

Simulated hashes (marked [sim]) are used when the block API is unreachable. These games cannot be verified on-chain.

How it works

01
A deciding hash picks who goes first
The last significant hex digit of a Bitcoin block hash determines first-mover. Odd digit → Bitcoin goes first. Even digit → Ethereum goes first.
02
Leading zeros are stripped
Bitcoin hashes begin with many zeros — these are structural (proof-of-work difficulty) and carry no randomness. Everything up to and not including the first non-zero character is stripped. From that first non-zero character onward, all characters — including any zeros that appear — are genuine random entropy and count normally.
03
Hash characters select the move
Characters 1-2 select which piece to move. Characters 3-4 select the destination square. If the result is an illegal move, the engine slides further along the hash until a legal move is found.
04
Chains alternate turns
Bitcoin makes a move, then Ethereum, then Bitcoin — regardless of each chain's block time. One move per block, per chain, per turn.
05
The game ends when chess ends
Checkmate, stalemate, or a 200-move draw. The winner is recorded. A new game begins.
